# 하네스 초기 설치 (init-harness)

meta_harness 스타터킷을 실제 프로젝트 하네스로 전환한다. **목표: 이 세션이 끝나면 `{{...}}` 플레이스홀더 0, 안 쓰는 팩 0, 킷 메타 파일 정리 완료.**

## 0. 설치 유형 판별

- **새 프로젝트**: 킷이 곧 루트 → 1번부터 순서대로.
- **기존 프로젝트에 도입(레트로핏)**: 기존 파일과 충돌 주의 —
  - 기존 `README.md`가 있으면 킷 README를 덮어쓰지 말 것 (킷 README·HARNESS.md는 참고 후 삭제, 필요 내용만 기존 문서에 병합).
  - 기존 `CLAUDE.md`가 있으면 **덮어쓰기 금지** — 킷 CLAUDE.md의 절 구조(§0~§8)를 기존 내용과 대조해 빠진 절만 추가.
  - 기존 `.claude/settings.json`이 있으면 permissions만 병합.

## 1. 프로젝트 조사 (문답 전에 — 물어볼 것을 줄인다)

- 스택·구조: package.json / 빌드 설정 / 디렉터리 구조 정독 → CLAUDE.md 머리말 초안.
- 검증 명령: 기존 스크립트(test/build/typecheck)를 실제 실행해 동작 확인 → §3 초안. 테스트가 있으면 현재 테스트 수 = 기준선 초기값.
- git 로그·기존 문서에서 프로젝트 정체성·진행 중 작업 파악.

## 2. 사용자 문답 (조사로 못 채우는 것만 — 한 번에 하나씩)

1. 프로젝트 한 줄 정의가 조사 결과와 맞는가? 확장 방향(좁게 오해하면 안 되는 것)은?
2. 도메인 불변식(§2): "절대 깨지면 안 되는 규칙"이 무엇인가? (재현성·데이터 규칙·사용자 적합성)
3. 불변 결정(§5): 이미 내려진, 되돌리면 안 되는 결정이 있는가?
4. 커밋 규약(§7): 커밋 메시지 언어·배포 트리거·커밋 금지 대상.
5. 팩 선택: 콘텐츠 대량 생산? → content-verification / 사용자 데이터 DB? → db-analytics / 민감 도메인? → security-privacy / 같은 파일 병렬 실험·코드리뷰? → code-collab.

## 3. 조립

1. CLAUDE.md 플레이스홀더 전부 채움 (해당 없는 절은 삭제 — 빈 절을 남기지 말 것).
2. 선택된 팩: 각 팩 `INSTALL.md` 절차 실행 (스킬/문서 제 위치 복사 + CLAUDE.md 절 추가). 미선택 팩: `packs/<이름>/` 폴더째 삭제.
3. `.claude/agents/README.md` 경계표를 실제 파일 구조로 채움 (에이전트가 아직 필요 없으면 표를 비우되 "공유 순차 파일" 목록은 지금 채워라 — 나중에 병렬 도입 시 가장 먼저 필요하다).
4. `docs/decisions-log-template.md` → `docs/decisions-log.md` 복사 후 §2 문답의 결정 기록.
5. 킷 메타 정리: `packs/` 잔여물·킷 README.md·HARNESS.md 처리 (새 프로젝트면 README는 프로젝트 소개로 교체, HARNESS.md는 docs/로 이동하거나 삭제 — repo 링크가 원본).

## 4. 설치 검증 (완료 선언 전)

- [ ] `node scripts/check-placeholders.mjs` → 0건 (docs/의 `*-template.md`류는 영구 템플릿이라 검사 제외가 정상)
- [ ] CLAUDE.md §3 명령을 **실제 실행**해 전부 동작 (안 도는 명령을 규약에 박으면 첫 커밋부터 하네스 불신)
- [ ] 참조 무결성: CLAUDE.md가 가리키는 파일 전부 존재
- [ ] 사용자에게 완성된 CLAUDE.md 검토 요청 → 승인 후 첫 커밋
- [ ] 이 스킬 폴더(`.claude/skills/init-harness/`) 삭제 (일회용)

## 합리화 방지

| 유혹 | 규율 |
|---|---|
| "플레이스홀더 몇 개는 나중에 채우지" | {{}} 잔존 = 다음 세션이 깨진 규약을 읽는다. 0건이 완료 조건. |
| "검증 명령은 안 돌려봐도 문서만 맞으면" | 실행 안 해본 명령을 규약화 금지 (4번). |
| "기존 CLAUDE.md 위에 킷 걸 덮어쓰자" | 레트로핏은 병합만 — 기존 규약은 실전 기록이다. |
